Installation

SimGrid Eclipse plug-in can be installed as any normal plug-in. The comlete process is quite simple:

  1. Select Help > Install New Software. The install dialog will appear.
  2. Type http://simgrideclipse.forge.imag.fr/eclipse/ in the "work with:" field
  3. Select the plugin
  4. Click on next until finished
Note that the SimGrid Eclipse Plugin is independent from SimGrid distribution. In order to use and compile your code, you have to install SimGrid. You can retrieve SimGrid distribution for your favorite OS on the SimGrid Website.
