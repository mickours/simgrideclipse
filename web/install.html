Installation

Install it into Eclipse

SimGrid Eclipse plug-in can be installed as any normal plug-in. The comlete process is quite simple:

  1. Select Help > Install New Software. The install dialog will appear.
  2. Type http://simgrideclipse.forge.imag.fr/eclipse/ in the "work with:" field
  3. Select the plugin
  4. Click on next until finished
Note that the SimGrid Eclipse Plugin is independent from SimGrid distribution. In order to use and compile your code, you have to install SimGrid. You can retrieve SimGrid distribution for your favorite OS on the SimGrid Website.

Requirements

This plugin was tested on the folowing Eclipse version and operating systems (OS).

Eclipse version:

  • Eclipse Indigo (3.7)
  • Eclipse Juno (4.2)
OS:
  • Linux
  • Windows XP
  • MacOSX

WARNING

Make sure that you have install ALL the dependencies for SimGrid depending on your OS. See SimGrid install instructions

Make sure that you have set ALL the following ENVIRONMENT VARIABLES:

  • SIMGRID_ROOT: your SimGrid root install directory
  • SIMGRID_JAVA_ROOT (only if you are using java bindings): your SimGrid Java root install directory
  • This variable is OS dependent, it contains paths for the needed libraries
    • LD_LIBRARY_PATH (Linux)
    • DYLD_LIBRARY_PATH (MacOSX)
    • PATH (Windows)
