Installation

Install it into Eclipse

SimGrid Eclipse plug-in can be installed as any normal plug-in. The complete process is quite simple:

  1. select Help > Install New Software. The install dialog will appear.
  2. type http://simgrideclipse.forge.imag.fr/eclipse/ in the "work with:" field
  3. select the plugin
  4. click on next until the installation begins
  5. you will be warn that the plug-in is not signed, click OK
  6. restart Eclipse and it works!
Note that the SimGrid Eclipse Plug-in is independent from SimGrid distribution. In order to use and compile your code, you have to install SimGrid. You can retrieve SimGrid distribution for your favorite OS on the SimGrid Website.

Requirements

To make this plug-in works you need an operating systems (OS) with Java 1.6 or later and Eclipse installed. You can download Eclipse here. This plug-in was tested on the following Eclipse version and OS.

Eclipse version

  • Eclipse Indigo (3.7)
  • Eclipse Juno (4.2)

WARNING

For each Eclipse version you need to have the official Eclipse Update Site in your "Available Software Site" (http://download.eclipse.org/releases/indigo/ or http://download.eclipse.org/releases/juno/).

OS

  • Linux
  • Windows XP
  • MacOSX

WARNING

Make sure that you have install ALL the dependencies for SimGrid depending on your OS. See SimGrid install instructions.

Make sure that you have set ALL the following ENVIRONMENT VARIABLES:

  • SIMGRID_ROOT: your SimGrid root install directory
  • SIMGRID_JAVA_ROOT (only if you are using java bindings): your SimGrid Java root install directory
  • This variable is OS dependent, it contains paths for the needed libraries
    • LD_LIBRARY_PATH (Linux)
    • DYLD_LIBRARY_PATH (MacOSX)
    • PATH (Windows)

on Linux and MacOSX

To set these variables on UNIX-Like OS (Linux, MacOSX), it is recommended to edit the ".profile" file in your home folder. Create it if it does not exists and add the following lines. Be careful, the path "/usr/local" is the default but you must change it if you have used a custom location.

on Linux

export SIMGRID_ROOT="/usr/local"
export SIMGRID_JAVA_ROOT="/usr/local"
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:$SIMGRID_ROOT/lib:$SIMGRID_JAVA_ROOT/java

on MacOSX

export SIMGRID_ROOT="/usr/local"
export SIMGRID_JAVA_ROOT="/usr/local"
export DYLD_LIBRARY_PATH=$DYLD_LIBRARY_PATH:$SIMGRID_ROOT/lib:$SIMGRID_JAVA_ROOT/java

on Windows

  1. from the Desktop, right-click on My Computer and select Properties OR in the Configuration Panel go into System
  2. click on Advanced System Settings tab
  3. in the System Properties window click on Environment Variables
  4. set the SIMGRID_ROOT, SIMGRID_JAVA_ROOT and PATH as the SimGrid installer said
