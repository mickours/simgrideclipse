Documentation

SimGrid eclipse plug-in adds the following functionalities to Eclipse:

  • A SimGrid MSG project Wizard that creates for you all files needed to start a MSG simulation, in both C and Java
  • A visual Editor for platform files that helps you to visualize and edit your platform files. To use it instead of the normal XML editor given by Eclipe, all you have to do is to right-click on your platform description‚Äç file and to select the SimGrid Editor.
